# 🎨 AI 마법 벡터 드로잉 스튜디오 Pro Ultra (v5.7)

> **"자유로운 픽셀 스케치와 정밀한 기하학적 벡터의 완벽한 융합."**
> 본 프로젝트는 웹 브라우저 환경에서 작동하는 고성능 하이브리드 드로잉 스튜디오입니다. 자유 스케치 드로잉 기능과 더불어 대형 마스터 도안 라이브러리 및 AI 패턴 매칭 엔진을 통해 초광활 시네마틱 캔버스 위에서 우주급 대작을 실시간으로 설계할 수 있습니다.

---

## 🚀 주요 특징 (Key Features)

* **📺 초광활 시네마틱 캔버스 (Cinema-Scope Canvas)**
    * 주변 여백과 불필요한 패딩을 싹 걷어내어 브라우저 가용 면적을 극한으로 넓혔습니다.
    * 화면 크기에 따라 실시간으로 해상도를 자동 동기화합니다.
* **📐 하이브리드 드로잉 매트릭스**
    * **자유 스케치 모드:** 픽셀 기반의 부드러운 핸드 드로잉 및 지우개 모드를 지원합니다.
    * **선 내부 채우기 (Flood-Fill):** 자유 곡선으로 가둬진 공간을 1픽셀의 오차 없이 깔끔하게 채웁니다.
    * **벡터 파츠 컬러링:** 기하학적 벡터 파츠의 내부를 클릭 한 번으로 스마트 채색합니다.
* **🤖 AI 고해상도 패턴 분석 엔진**
    * 그려진 스케치의 흐름을 분석하여 가장 어울리는 추천 도안 파츠 구조를 강제로 융합·병합합니다.
* **🎛️ 정밀 레이어 제어반 & 플로팅 HUD**
    * 각 파츠별 위치($X$, $Y$), 회전 각도, 회전 피벗(Pivot) 중심점 설정, 상하좌우 반전(Flip Matrix)을 실시간 수치로 완벽하게 제어합니다.
    * 특정 파츠만 분리하여 볼 수 있는 **'단독 격리 모드(Isolation Mode)'**를 탑재했습니다.
* **⚔️ 마스터 도안 라이브러리 (20종 완비)**
    * 네온 기샤, 안드로이드의 눈물, 달을 삼킨 늑대, 단청 잉어 역류, 아르누보 넝쿨 열쇠 등 감각적이고 독창적인 20가지의 멀티 파츠 프리셋 테마를 기본 제공합니다.
    * 새로 추가되는 도안은 겹침 없이 안전하게 자동 분산 배치됩니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS ES6+)
* **Styling:** Tailwind CSS (CDN), Font Awesome 6.4.0
* **Engine:** HTML5 2D Context Canvas API Engine
* **Fonts:** Pretendard, Monospace

## 🖥️ 미리보기 및 인터페이스 (Interface Overview)

| 컴포넌트 | 기능 및 특징 |
| :--- | :--- |
| **마스터 제어반 (좌측)** | 드로잉 모드 변경, 익스프레스 컬러 스와치, RGB 배합 컬러칩 추가, 붓 두께 제어, 도안 라이브러리 탑재 |
